#### StringBuilder 클래스를 사용한 reverse
```java
    String a = "abcde";
    String str = new StringBuilder(a).reverse.toString();
    System.out.println(str);
```
결과값은 edcba가 나오게 된다.

<br>

#### while로 구현

```java
    
    String a = "abcde";
    char[] s = a.toCharArray();
    int lt = 0, rt = a.length()-1
    while(lt < rt){
        char tmp = s[lt];
        s[lt] = s[rt];
        s[rt] = tmp;
        lt++;
        rt--;
    }
    
```
s를 출력했을 때 edcba가 나옵니다.
