JSP(Java Server Pages)
===

<br>
Serverside-java-template

HTML 코드에 자바코드를 넣어 동적인 웹페이지를 구성할 수 있도록 해준다.

Servlet을 보완한 형태이며, Servlet의 모든기능과 더불어 부가기능을 가진다.
```
<%= expression %>
```
```
<% code %>
```
자바 코드를 작성할 수 있다.
```
<%! declaration %>
```
```
<%-- comment --%>
```
```
<%@ directive %>
```
